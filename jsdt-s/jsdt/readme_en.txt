Javascript Debug Toolkit(JSDT) is a javascript debug tool.
JSDT can debug javascript in any browser support Ajax, like
    IE6
    IE7,
    IE8,
    Firefox1.5,
    Firefox2,
    Firefox3,
    Safari,
    Chrome,
    Opera,
and some mobile browser.
JSDT can run in windows and linux OS.
JSDT is available under the LGPL.
