Javascript Debug Toolkit(JSDT) 是一款免费绿色免安装的javascript调试工具，
JSDT可以在任何支持ajax的浏览器上调试javascript，譬如:
    IE6
    IE7,
    IE8,
    Firefox1.5,
    Firefox2,
    Firefox3,
    Safari,
    Chrome,
    Opera,
和一些 mobile browser.
JSDT 可以运行在 windows 和 linux 操作系统上.
JSDT遵守GPL或LGPL开源协议.
